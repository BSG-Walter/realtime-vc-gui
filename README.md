## Usage
You can download the latest release from **[here](https://nightly.link/fishaudio/realtime-vc-gui/workflows/ci/main)** 

or install from PyPI.

```bash
pip install rtvc
rtvc
```

or use pipx

```bash
# or
pipx run rtvc
```
